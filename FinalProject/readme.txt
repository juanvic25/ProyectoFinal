La Pagina Web es para registrar Reseñas de Peliculas, a continuación se muestran las Reglas de la Web:

MENU:
INICIO: muestra toda la lista de Peliculas
CATEGORIAS: muestra la lista de peliculas de la categoria seleccionada
REGISTRAR / Categorias: permite registrar o editar Categorias
REGISTRAR / PEliculas: permite registrar nuevas peliculas

Si no estas logueado:
1. Puedes ver el listado de peliculas
2. Puedes ver el listado de Reseñas por cada pelicula
3. No Puedes registrar Reseñas de Peliculas
4. No puedes registrar o editar Peliculas o Categorias
5. Puedes Registrar tu Usuario de la Web

Si estas logueado como Usuario:
1. Puedes ver el listado de peliculas
2. Puedes ver el listado de Reseñas por cada pelicula
3. Puedes registrar Reseñas de Peliculas (solo 1 reseña por pelicula)
4. Puedes editar o eliminar unicamente tus reseñas
5. No puedes registrar o editar Peliculas o Categorias
6. Puedes editar los datos de tu perfil
7. Puedes cambiar tu contraseña

Si estas logueado como Administrador:
1. Puedes ver el listado de peliculas
2. Puedes ver el listado de Reseñas por cada pelicula
3. No Puedes registrar Reseñas de Peliculas
4. Puedes eliminar cualquier reseña
5. Puedes registrar o editar cualquier Peliculas
6. Puedes registrar o editar cualquier categoria (Se actualiza automaticamente el listado que se muestra en el Menu)
7. Puedes editar los datos de tu perfil
8. Puedes cambiar tu contraseña
